# Machine Learning Clustering Analysis

This project implements clustering techniques using Python and scikit-learn.  
The main focus is on applying the K-Means algorithm, evaluating clustering performance, and analyzing results.

---

## 📌 Project Overview

In this notebook, we:

- Load and preprocess the dataset
- Apply K-Means clustering
- Compute evaluation metrics such as SSE
- Analyze cluster centroids
- Visualize clustering results

This project demonstrates understanding of unsupervised learning concepts and practical implementation.

---

## 🧠 Algorithms Used

- K-Means Clustering
- Centroid computation
- Sum of Squared Errors (SSE)
